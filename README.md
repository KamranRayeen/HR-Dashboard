# Human Resources Dashboard (Tableau)

## Project Overview
This project showcases an interactive HR Dashboard built using Tableau to analyze employee data across multiple dimensions such as demographics, performance, and salary.

It provides both high-level summaries and detailed insights to support HR decision-making.

---

## Objectives
- Track employee lifecycle (Hired, Active, Terminated)
- Analyze workforce demographics
- Understand education vs performance trends
- Evaluate salary distribution
- Identify workforce patterns and insights

---

## Dashboard Features

### 🔹 Overview
- Total Active Employees
- Hired vs Terminated trends
- Department-wise distribution
- HQ vs Branch comparison
- Location-based visualization

### 🔹 Demographics
- Gender distribution
- Age group analysis
- Education level breakdown
- Education vs performance insights

### 🔹 Income Analysis
- Salary comparison by education & gender
- Age vs salary correlation
- Role-based salary insights
